# 5opkaWebsite

**5opkaWebsite** — это веб-сайт посвященный творчеству и деятельности великого и ужасного контент-мейкера 5opka, на сайте присутсвуют несколько страниц, каждая из которых репрезентует его с разных сторон: биография, последние события (новости), музыка и галерея. Сайт построен на Python с использвоанием фреймворка Bottle, CSS и JavaScript.

## Начало работы

Эти инструкции помогут вам получить копию проекта и запустить его на локальной машине для целей разработки и тестирования. 

### Предварительные требования

Что необходимо установить для работы с проектом:

*   **Git** — для клонирования репозитория.
*   **Python** (версия 3.x) — среда выполнения.
*   **pip** — менеджер пакетов Python (обычно идёт вместе с Python).

Убедитесь, что Python и pip установлены:
```bash
python --version
pip --version
```

### Установка

Пошаговая инструкция по настройке среды разработки:

1.  **Клонируйте репозиторий** и перейдите в папку проекта:
    ```bash
    git clone https://github.com/NTC539/5opka.git
    cd 5opka
    ```

2.  **Создайте виртуальное окружение** (рекомендуется для изоляции зависимостей):
    ```bash
    python -m venv venv
    ```

3.  **Активируйте виртуальное окружение**:
    *   На Windows:
        ```bash
        venv\Scripts\activate
        ```

4.  **Установите зависимости** из файла `requirements.txt`:
    ```bash
    pip install -r requirements.txt
    ```

5.  **Запустите приложение**:
    ```bash
    python 5opkaWebsite/app.py
    ```

После успешного запуска сервер обычно становится доступен по адресу `http://localhost:8080` (порт может отличаться — проверьте отладочную консоль).
